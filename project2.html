Star Compass

Embedded C / Arduino Sensors Embedded Systems Signal Processing and Filtering Electronics

Description

This project is a handheld Arduino-based astrological compass that guides users toward celestial bodies using real-time sensor fusion. It integrates a GPS module, accelerometer, magnetometer, and real-time clock to calculate the user's position, orientation, and local sky coordinates via the SkyMap library. Users can select a target on an LCD interface, and the device continuously updates its directional guidance by comparing sensor data to the object's azimuth and altitude. The system combines sensor calibration, noise handling, and embedded programming to deliver an accurate and interactive astronomical navigation tool.
